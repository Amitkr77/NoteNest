# NoteNest

NoteNest is a simple web application for managing your notes and tasks. This project uses HTML, CSS, and JavaScript to provide an easy-to-use interface for adding, viewing, and deleting notes. Notes are stored in the browser's local storage, allowing them to persist between sessions.

## Features

- **Responsive Design**: The application is designed to be responsive and works well on both desktop and mobile devices.
- **Add Notes**: Users can add a note with a title and description.
- **Delete Notes**: Users can delete individual notes.
- **Clear All Notes**: Users can clear all notes from the local storage.
- **Dropdown Menu**: Contains additional options.
- **Search Functionality**: A search bar is available for future implementation.

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript (ES6)

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

You need a modern web browser and optionally a local server to handle the files more efficiently.

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/your_username/NoteNest.git
   ```
2. Open the project directory
   ```sh
   cd NoteNest
   ```
3. Open `index.html` in your browser to view the application.

## Usage

### Adding a Note

1. Enter a title in the "Work" input field.
2. Enter a description in the "Your Work Description" textarea.
3. Click the "Add to Your Work Book" button to save the note.

### Deleting a Note

1. Click the "Delete" button next to the note you want to remove.

### Clearing All Notes

1. Click the "Clear Work Book" button.
2. Confirm the action in the prompt that appears.

### Dropdown Menu

1. Click on the "items" button to toggle the dropdown menu.
2. Click outside the dropdown to close it.
